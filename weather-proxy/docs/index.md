# Weather Proxy Documentation

Welcome to the Weather Proxy documentation!

## Overview

The Weather Proxy service provides a simple API to fetch weather data.

## API

### GET /weather

Returns the current weather for a hardcoded location.

**Response:**

```json
{
  "location": "Stockholm",
  "temperature": 20,
  "condition": "Sunny"
}
```
